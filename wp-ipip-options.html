WP-IPIP

简介

WP-IPIP 是一款根据 IP 显示评论来源地址的插件。


设置

IPIP.net 数据库文件

文件状态：{{$ipip_data_status}}

说明：默认 IPIP 的数据库文件存放于插件目录下的{{$ipip_data_filepath}}目录下，文件名为{{$ipip_data_filename}}。

您可以到 IPIP.net 官网 下载更新的数据库文件，并上传到插件目录下的{{$ipip_data_filepath}}目录下。

项目地址

GitHub：https://github.com/catscarlet/wp-ipip

本项目暂未包含于 WordPress Plugin 仓库中，因此不支持自动更新。点击 这里 下载最新的版本
